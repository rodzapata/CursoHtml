    Metodo abreviado en Netbeans

    En este blog aparecen los teclado abreviado para Netbeans panamahitek.com
    Tambien podemos encontrar mas atajos de teclado en este blog byspel.com

    los siguienes son metodo abreviado en Neabens
    sout + Tab escribe : System.out.println("");
    Shif + Alt + flecha abajo o arriba, mueve el texto selecionado hacia abajo
    crl + Shif + flecha abajo Duplica la linea.
    crl + Shif + C : coloca en comentario el texto seleccionado.
    main + Tab Crea el metodo main de Java
    for + Tab ciclo for del 1 al 10
    fori + Tab ciclo for para array
    pr + Tab private
    wh + Tab ciclo While
    st + Tab palabra static
    sw + Tab switch(
    sy + tab synchronized

    alt + Scroll de mouse, Agranda y empequeñeze el tamaño de la fuente del editor

    para habilitar autocompletar de codigo en Neatbeans, hay que habilitalo en la ruta Tool - option - Petaña Editor - Code Completacion - Language : escojemos Java
    y activamos las casillas Guess Filed Method Arguments, tambien debemos activar
    la casilla Auto Popud on Typing Any Java Identifier Part
