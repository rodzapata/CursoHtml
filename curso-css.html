    Selectores

    existen tres tipos de selectores Básicos: (etiquetas o tipos, identificadores y Clases) las etiquetas son seleccionado por el nombre de los diferentes elementos html,
    los identificadores hay que seleccionarlos por el atributo id-#, Las Clases hay que seleccionarlas por el atributo . (punto)
    el selector universal es el * ( asteriscos)

    Selectores de hermanos generales

    un elemento especifico que esté inmediatamente despues de otro elemento especificado.
    selector general de hermanos se utliza el simbolo ~ 
            Sintaxis: A ~ B
        Ejemplo: La regla p ~ span se aplicará a todos los elementos <span> que siguen un elemento <p>.
        p ~ span {
            color: red;
        }

    Selectores de hijos

    El combinador > selecciona los elementos que son hijos directos del primer elemento. 
            Sintaxis: A > B
        
        span { background-color: white; }
            div > span {
            background-color: DodgerBlue;
        }
    Span #1, dentro del div. Span #2, dentro del span que está en el div.
    Span #3, no está dentro del div.

    Bordes (estilos)

    Es el recuardo o cuadro que embarca un objeto, existen diferentes estilos como son:
    none, double,groove, dotted, inset, solid, dashed
    donde none es sin bordes,
    double coloca un recuadro de doble linea,
    groove ranura, linea delgada
    dotted punteado,
    inset inserta un recuadro con el borde izquierdo y superior oscuro en contraste con un border inferior y derecho mas claro.
    solid es un recuadro totalemente oscuro, linea gruesa, sin ningun tipo de constrate.
    dashed bordes con guiones, rayado.
    ejemplo :

    ejemplo de bordes doble

            Codigo Html:
  
        <h3 id="borde-ejemplo1">ejemplo de bordes doble</h3>
  
        Codigo CSS:
  
        #borde-ejemplo1{
          border-style: double;
        }
    es posible combinar varios estilos, como son cutro bodes se colocan en direccion a las manecillas del reloj ose :
    border superior, derecha, inferior, izquieda.

    ejemplo de bordes combinado

            Codigo HTMl:
        <h3 id="borde-combinado">ejemplo de bordes combinados</h3>
  
        codigo CSS:
        #borde-combinado{
          border-style: double dotted solid none;
        }
    Tambien podemos agregarles mas propiedades a los bordes como el color, o el radio de los

    Color en los enlace

    cuando se ha visitado un sitio que tiene un HREF, ya se aun <a> o un <link>,
    cambia el color del enlace esto lo podemos personalizar con hojas de estilos.
    por ejemplo si no se ha visitado el enlace que sea de color azul violeta 
            
          ejemplo CSS enlace sin visitar
          a:link{
            color: blueviolet;
          }
    en caso que se haya visitado el enlace que se coloque de color gris .
            
          ejemplo CSS enlace visitado
          a:visited{
            color: gray;
          }
    cuando el usuario se coloca sobre un objeto de enlace, pero sin darle clik cambia el color del objeto a color amarillo verdoso.
            
          ejemplo CSS
          a:hover{
            color: yellowgreen;
          }
    si el enlace está activo, que se vea de color rojo. 
            
          ejemplo CSS enlace activo
          a:active{
            color: red;
          }

    Display

    especifica si un elemento es tratado como block ose uno debajo de otro, o Inline, un elemento al lado del otro.
    y el diseño utilizado para sus hijos, como flow layout (Diseño de flujo), grid (Cuadricula) o flex flexible.
    la propiedad Display establece dos tipos de visualizacion: interna y externa de un elemento.
    la externa es como se comporta con los otros elementos, y la interna es como se comporta con sus elementos hijos.
    A los elementos se les puede dar un ancho maximo con la propiedad max-with y tambien se le puede establecer una posicion.
    la position puede tener los siguientes valores: static, relative, absolute, fixed 
            
          ejemplo css
          .parrafo-fijo{
            background-color: aqua;
            position: fixed;
            max-width: 344px;
            left: 25px;
            top: -15px;
          }

    Maquetacion estilo Flex Box

    Este estilo el primer estilo de maquetado. y todavia se manteniene. existe una pagina recomendada que se llamaba:
    CSS trick, es muy pero muy recomendada.
    Tambien es muy recomendable descargar la libreria CSS Flexbox Cheatsheet desde visual studio Code.
    para trabajar se debe utilizar la propiedad display, que puede tener los valor de flex, inline-flex donde flex, funciona en bloque, y inline-flex, ocupa varios en una misma linea.
    la propiedad flex-direction, nos indica la direccion en la que aparecen los item:
    row, row-reverse, column, column-reverse. el row es el valor por defecto y la direccion es de izquierda a derecha,
    row-reverse, es lo contrario es decir direccion de derecha a izquierda., column tiene la direccion de arriba hacia
    abajo y column-reverse seria de abajo hacia arriba.
    propiedad flex-wrap, nos permite distribuir una linea en varias, y pueden ser :
    nowrap no envolvente, es el valor por defecto, wrap una o mulitiples lineas de arriba hacia abajo,
    wrap-reverseuna o multiples lineas de abajo hacia arriba.

    Existe la propiedad Flex-flow que combina la dos propiedades anteriores (flex-direction y flex-wrap) los valores por defecto son row y nowrap simultaneamente.

    Alinear espacios entre item

    Una característica clave de flexbox es la capacidad de alinear y justificar ítems sobre los ejes principal y cruzado, y distribuir el espacio entre los ítems flex.
    justify-content, define la alineacion de los item hijos.
    sus valores son: center, flex-start,flex-end, space-between, space-arroud, space evenly. Esta propieda alinea los elemento en el eje X, por lo tanto para que se note los cambios, hay que tener la propiedad
    flex-direction en filas (row) y flex-wrap como wrap para que ocupe una o mas lineas.
    alig-item, esta propiedad alinea los items sobre el eje cruzado osea el eje de las Y.
    el valor por defecto es stretch razón por la cual los ítems se ajustan por defecto a la altura de aquel más alto.
    tambien puede tener los valor flex-start, flex-end, center, donde se alineas los item al comienzo, al final o centrado,
    con respecto al contenedor.
    Para poder trabajar con esta propiedad tenemos que tener el contenedor en columnas, osea flex-direction=column y flex-wrap=wrap para que los item de los hijos se alarguen con respecto al contenedor, deber tener un tamaño ya se en alto o en ancho.
    si los item estan alineado en columnas, el contenedor tiene que tener un alto height y si esta alineado a en fila,
    el contenedor debe tener un ancho. width.

    Grilla

    La grilla complementa al estilo de programacion de antes que se llamaba Flex box. A partir del 2014
    la grilla o Grid se arma usando Div y se combina con la propiedad Display=Grid
    con la propiedad grid-template-columns: podemos establecer el tamaño de las columnas, en este caso serian 3 columnas
    de tamaño 60px 120px 30px, es caso que el contenido del texto no cabe en el tamaño de la columna asignada se recorta el contenido del texto.
    grid-template-rows:20px ; estable un alto de la linea de 20 pixeles. Tambien se puede utlizar fr que seria fracccion,
    por ejemplo que la segunda columna sea el doble de tamaño de la primera y la tercera sea tres veces mas ancha que la primera, asi:
    grid-template-columns: 1fr 2fr 3fr; en caso que queramos que las 3 columnas sean del mismo ancho se puede utiliar la propiedad
    Repeat de la siguiente manera: grid-template-columns: repeat(3, 1fr);
    Tambien se puede establecer el tamaño mínimo y máximo de una columna o una fila con la funcion minmax()
    por ejemplo podemos decir que una columna tenga de tamaño como minimo 50px y maximo 230px quedando el codigo asi: 
            
          .grilla{
            display: grid;
            grid-template-columns: 60px 120px 30px;
            grid-template-columns: minmax(50px, 230px);
            grid-template-rows:20px ;
            grid-row-gap: 10px;
            grid-column-gap: 10px;
       
          }
          .element{
            background-color: aqua;
          }
