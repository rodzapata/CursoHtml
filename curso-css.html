    Selectores

    los identificadores hay que seleccionarlos por el atributo id-#, Las Clases hay que seleccionarlas por el atributo . (punto)

    Selectores de hermanos generales

    un elemento especifico que esté inmediatamente despues de otro elemento especificado.
    selector general de hermanos se utliza el simbolo ~ 
          Sintaxis: A ~ B
      Ejemplo: La regla p ~ span se aplicará a todos los elementos <span> que siguen un elemento <p>.
      p ~ span {
          color: red;
      }

    Este span no es rojo.

    Selectores de hijos

    El combinador > selecciona los elementos que son hijos directos del primer elemento. 
          Sintaxis: A > B
      
      span { background-color: white; }
          div > span {
          background-color: DodgerBlue;
      }
    Span #1, dentro del div. Span #2, dentro del span que está en el div.
    Span #3, no está dentro del div.

    Selectores de descendiente

    se utiliza el espacio
          Sintaxis: A B
      
      span { background-color: white; }
          div span { background-color: DodgerBlue; }

    Bordes (estilos)

    Es el recuardo o cuadro que embarca un objeto, existen diferentes estilos como son:
    none, double,groove, dotted, inset, solid, dashed
    donde none es sin bordes,
    double coloca un recuadro de doble linea,
    groove ranura, linea delgada
    dotted punteado,
    inset inserta un recuadro con el borde izquierdo y superior oscuro en contraste con un border inferior y derecho mas claro.
    solid es un recuadro totalemente oscuro, linea gruesa, sin ningun tipo de constrate.
    dashed bordes con guiones, rayado.
    ejemplo :

    ejemplo de bordes doble

          Codigo Html:

      <h3 id="borde-ejemplo1">ejemplo de bordes doble</h3>

      Codigo CSS:

      #borde-ejemplo1{
        border-style: double;
      }
    es posible combinar varios estilos, como son cutro bodes se colocan en direccion a las manecillas del reloj ose :
    border superior, derecha, inferior, izquieda.

    ejemplo de bordes combinado

          Codigo HTMl:
      <h3 id="borde-combinado">ejemplo de bordes combinados</h3>

      codigo CSS:
      #borde-combinado{
        border-style: double dotted solid none;
      }
    Tambien podemos agregarles mas propiedades a los bordes como el color, o el radio de los

    ejemplo de bordes redondeados, color

          Codigo HTMl:
      <h3 id="borde-rendondeados">ejemplo de bordes redondeados, color</h3>

      codigo CSS:
      #borde-rendondeados{
        border-style: solid;
        border-color: blue;
        border-radius:5px ;
        border-width: 5px;
      }

    Color en los enlace

    cuando se ha visitado un sitio que tiene un HREF, ya se aun <a> o un <link>,
    cambia el color del enlace esto lo podemos personalizar con hojas de estilos.
    por ejemplo si no se ha visitado el enlace que sea de color azul violeta 
          
        ejemplo CSS enlace sin visitar
        a:link{
          color: blueviolet;
        }
    en caso que se haya visitado el enlace que se coloque de color gris .
          
        ejemplo CSS enlace visitado
        a:visited{
          color: gray;
        }
    cuando el usuario se coloca sobre un objeto de enlace, pero sin darle clik cambia el color del objeto a color amarillo verdoso.
          
        ejemplo CSS
        a:hover{
          color: yellowgreen;
        }
    si el enlace está activo, que se vea de color rojo. 
          
        ejemplo CSS enlace activo
        a:active{
          color: red;
        }

    estilos en las tablas

    la propiedad border-collapse, es utlizadas para unir o separar los bordes, collapse unidos o separate separados.
    :nth-child funcion que nos permite seleccionar una condicion de los objetos hermanos, donde odd son los elementos impares
    y even son los elementos pares dentro de los registros de una tabla.
    en ese ejemplo los item paren van ha tener el color verde como se muestra en el siguiente ejemplo: 
          
        ejemplo css
        .tabla-familia tr:nth-child(even){
          background-color: green;
        }
    Tambien se puede utilizar Hover para que cambie de color gris cuando seleccionamos un registro de la tabla.
          
        Ejemplo CSS
        .tabla-familia tr:hover{
          background-color: gray;
        }
    ejemplo de :nth-child 
          
        .tabla-familia tr:nth-child(even){
          background-color: green;
        }

    Display

    especifica si un elemento es tratado como block ose uno debajo de otro, o Inline, un elemento al lado del otro.
    y el diseño utilizado para sus hijos, como flow layout (Diseño de flujo), grid (Cuadricula) o flex flexible.
    la propiedad Display establece dos tipos de visualizacion: interna y externa de un elemento.
    la externa es como se comporta con los otros elementos, y la interna es como se comporta con sus elementos hijos.

    Grilla

    la grilla o Grid se arma usando Div y se combina con la propiedad Display=Grid
    con la propiedad grid-template-columns: podemos establecer el tamaño de las columnas, en este caso serian 3 columnas
    de tamaño 60px 120px 30px, es caso que el contenido del texto no cabe en el tamaño de la columna asignada se recorta el contenido del texto.
    grid-template-rows:20px ; estable un alto de la linea de 20 pixeles. Tambien se puede utlizar fr que seria fracccion,
    por ejemplo que la segunda columna sea el doble de tamaño de la primera y la tercera sea tres veces mas ancha que la primera, asi:
    grid-template-columns: 1fr 2fr 3fr; en caso que queramos que las 3 columnas sean del mismo ancho se puede utiliar la propiedad
    Repeat de la siguiente manera: grid-template-columns: repeat(3, 1fr); 
          
        .grilla{
          display: grid;
          grid-template-columns: 60px 120px 30px;
          grid-template-rows:20px ;
        }
        .element{
          background-color: aqua;
        }
